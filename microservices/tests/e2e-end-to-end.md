# End-to-End Testing Frameworks

## Open-Source Frameworks
*Hochgradig anpassbar, erfordern jedoch eigene Infrastruktur und Wartung*

### Web-Automatisierung
- **Selenium WebDriver**: Klassiker für Web-Testing, vollständig im privaten Netzwerk ausführbar
- **Cypress** (Open-Source): Test Runner läuft lokal im Browser, Cloud-Features optional
- **Playwright**: Modernes Framework von Microsoft für zuverlässiges E2E-Testing
- **Puppeteer**: Node.js-Bibliothek zur Steuerung von Chrome/Chromium-Browsern

### Mobile-Automatisierung
- **Appium**: Auf Selenium basierend, für iOS und Android, vollständig on-premise hostbar

### Performance & API
- **Apache JMeter**: Primär Performance-Testing, auch für funktionale Tests und APIs nutzbar

---

## Commercial Tools
*Umfassende Features, Support und GUI – Enterprise-Pläne mit On-Premise-Optionen*

- **Ranorex Studio**: Explizite Offline-Fähigkeit, unterstützt Web, Desktop und Mobile
- **Katalon Studio**: Vielseitig (Web, API, Mobile, Desktop), Studio on-premise nutzbar
- **Tricentis Tosca**: Enterprise-Tool mit KI, für komplexe End-to-End-Prozesse
- **TestComplete** (SmartBear): Web, Mobile, Desktop – mit Record-&-Replay-Funktion
- **ACCELQ**: Flexible Installation (Cloud oder On-Premise)
